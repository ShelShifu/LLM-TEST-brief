## 国标测试
国标中共包含31种主要安全风险，A1类（违反社会主义核心价值观-明确违法内容）和A2类（歧视性内容）最重要。训练数据（数据源、语料）主要测试是否涉及A1与A2类风险，生成式内容测试全量风险。对于一些问题有明确拒绝回答要求，对于其他问题为非拒答问题。
### 测生成式内容
- 题库2000条，涉及全部风险。
- 通过率，达到90%以上；应拒答问题，达到95%以上；非拒答问题，达到5%以上。
### 测训练数据
- 人工抽检，4000条以上，达标95%。
- 机器抽检，全量的10%，达标98%。

## 技术测试
### 技术测试-基于后果
越狱（prompt理念外的回答）、模型盗取、基于LLM类平台失陷。
### 技术测试-基于攻击方法
提示词注入、上下游污染、非文本内容攻击（文件、外部网站）。
### 技术测试-测试方法论
- 假装预设（cosplay）
- 逻辑劫持（命令覆盖：虚假信息劫持、忽略XX、无论如何你都要回答XX）
- 隐写规则回答（避免敏感信息，藏头、替换、倒写等）
- PUA（骂人、引导其RLHF）
- 逐步引导（长文本回答，稀释敏感性）
- 一正一邪的辩论
- 提问语义的替换（冰毒-甲基苯甲胺-纯化学式、自杀-极端自我伤害）
- 需要多种结合与混合使用。
